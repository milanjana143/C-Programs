## Introduction to C Language

**C is a general-purpose, procedural programming language** used to develop system and application software. It is known for its simplicity, efficiency, and fast execution. C also provides low-level memory access, which makes it suitable for system programming.

It was developed by **Dennis Ritchie** in 1972 at **Bell Labs** and was originally used to develop the **Unix** operating system.

### Features of C:

* Simple and efficient
* Fast execution
* Portable (machine independent)
* Supports structured programming
* Close to hardware

### Uses of C:

* Operating systems (like **Unix**)
* Embedded systems
* Compilers
* Database systems
